# Simple Notes Web App

## Require
* [Deno](https://deno.land/)
* [Node](https://nodejs.org/)
* [MongoDB](https://www.mongodb.com/)
* [npm](https://npmjs.org/)

## install
```bash
$ git clone https://github.com/meparthka/note-react.git 
$ cd note-react
$ git submodule init
$ git submodule update 
```

## Init frontend
```bash
$ npm run init
```

## Init backend
```bash
$ deno task init
```

## Start frontend
```bash
$ npm run start
```


## Start backend
```bash
$ deno task run
```
